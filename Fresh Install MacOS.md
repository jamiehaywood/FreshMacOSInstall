# Things I want on a fresh install of OS X

Once in a while I like to create a clean install on my Mac. This is a list of all the software that I have on my clean image.

### Development Tools:
1. **Visual Studio Code** - Very powerful IDE for webdev and pretty much anything
1. **Git** - Preferred source control
1. **Node** - A must have for webdev
1. **Sublime Text 3** - Great text editor
1. **Parallels Desktop** - Parallelisation for MacOS
1. **Windows 10 LTSB** - Long Term Service Branch for no bloatware
1. **Visual Studio 2017** - A must have for .NET devs
1. **Dash** - Documentation for pretty much everything
1. **Tower** - Great way to visualise the git tree
1. **MacDown** - Top notch markdown editor
1. **Postman** - A must have if you're working with APIs
1. **Slack** - My preferred platform for communicating with other devs
1. **ForkLift** - Best FTP client by benchmarks

### Design Tools:
1. **Skala Preview** - Send live Photoshop renders to iOS devices
1. **Photoshop CS6** - The best image editing tool
1. **Vector Magic** - Great for a quick drag and drop vectorisation
1. **Adobe XD** - Mockup App with a very quick learning curve


### Productivity:
1. **Google Drive** - Great for cloud storage
1. **Google Chrome** - Preferred web browser
1. **Microsoft Word** - Word processor
1. **Microsoft Excel** - Table processor
1. **Paste** - Without a doubt my favourite productivity tool. Saves all things copied to the clipboard.
1. **Itsycal** - Awesome macOS toolbar utility that displays date and time in the toolbar. On click shows a mini calendar. "HH:mm E d MMM" <-- my display preferences.
1. **Mendeley** - Hands down the best citation software, with top notch integration with Word.
1.  

### Other Utilities:
1. **DiskInventoryX** - Visualise the space on your mac
1. **CleanMyMac** - Great for clearing out caches, unnecessary system files etc.
1. **TeamViewer** - For when your parents need remote support...
1. **TunnelBlick** - A must have OpenVPN client
1. **Transmission** - Fantastic lightweight torrent client
1. **VLC** - Without a doubt the best MacOS media player
1. **Unsplash** - Some fantastic images for the desktop. New every morning
1. **NordVPN** - Good for unblocking sites around the world
1. **Calibre** - Top notch eBook management software. Great for loading books onto Kindle.
1. **WiFi Spoof** - Good for those paid WiFi hotspots that give you 15 mins free.
1. **Spotify** - Favourite music streaming app.
1. **CarbonCopyCloner** - Great for creating an exact clone of my desktop.

# Chrome Addons
These chrome addons massively improve my productivity and provide me with useful information when browsing the web:

1. **The Great Suspender** - This suspends any tabs inactive for more than 5 mins. Great for reducing Chrome's memory overhead
1. **JSON Formatter** - Great for formatting JSON in browser
1. **Speedtest by Ookla** - Favourite speedtest tool
1. **Wappalyzer** - Identify the stack used by websites
1. **uBlock Origin & Extra** - Best in class adblocking addon
1. **Mendeley Importer** - Great for importing citation of websites/papers
