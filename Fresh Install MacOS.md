# Things I want on a fresh install of OS X

Once in a while I like to create a clean install on my Mac. This is a list of all the software that I have on my clean image.

### Development Tools:
1. **Visual Studio Code** - Very powerful IDE for webdev and pretty much anything
1. **Git** - Preferred source control
1. **Node** - A must have for webdev
1. **Sublime Text 3** - Great text editor
1. **Parallels Desktop** - Parallelisation for MacOS
1. **Windows 10 LTSB** - Long Term Service Branch for no bloatware
1. **Visual Studio 2017** - A must have for .Net devs
1. **Dash** - Documentation for pretty much everything
1. **Tower** - Great way to visualise the git tree
1. **MacDown** - Top notch markdown editor
1. **Postman** - A must have if you're working with APIs

### Design Tools:
1. **Skala Preview** - Send live Photoshop renders to iOS devices
1. **Photoshop CS6** - The best image editing tool
1. **Vector Magic** - Great for a quick drag and drop vectorisation
1. **Adobe XD** - Mockup App with a very quick learning curve


### Productivity:
1. **Google Drive** - Great for cloud storage
1. **Google Chrome** - Preferred web browser
1. **Microsoft Word** - Word processor
1. **Microsoft Excel** - Table processor

### Misc Utilities:
1. **DiskInventoryX** - Visualise the space on your mac
1. **ForkLift** - Best FTP client by benchmarks
1. **TeamViewer** - For when your parents need remote support...
1. **TunnelBlick** - A must have OpenVPN client
1. **Transmission** - Fantastic lightweight torrent client
1. **VLC** - Without a doubt the best MacOS media player
